# Tower Defense + Camp

Uruchomienie lokalnie:
1. Stwórz virtualenv:
   python -m venv .venv
   & .\.venv\Scripts\Activate.ps1   # Windows PowerShell
2. Zainstaluj:
   python -m pip install -r requirements.txt
3. Uruchom:
   python run_app.py
4. Otwórz: http://127.0.0.1:5000

Wyłączenie: Ctrl + C
